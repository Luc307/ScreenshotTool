I started by watching a YouTube Tutorial titeld "WPF Tutorial StackPanels".

- Layout panles are containers in which elements like Buttons can be placed
- The layout panel defines how the elements arrange
- The StackPanel stacks elements, the orientation attribute says if the elements are stacked vertically or horizontaly

The next Tutorial was "WPF Tutorial Gird"

- The Gird layout panel divides your window into columns and rows
- With each <RowDefinition/> element in <Grid.RowDefinitions> a row is created, same story goes for Columns
- The width of columns and the height of rows are set in the definiton Element, auto uses no more than the needed space for the content
- After the definitions start the actual elements
- Following creates an 9:9 Grid, places the elements with Grid.Row and Grid.Column and adjustes the span with Grid.RowSpan or Grid.ColumnSpan

<Grid>
 	<Grid.RowDefinitions>
 		<RowDefinition/>
 		<RowDefinition/>
 		<RowDefinition/>
 	</Grid.RowDefinitions>
 	<Grid.ColumnDefinitions>
 		<ColumnDefinition/>
 		<ColumnDefinition/>
 		<ColumnDefinition/>
 	</Grid.ColumnDefinitions>

 	<Button Grid.Row="0" Grid.Column="0">Button 0:0</Button>
 	<Button Grid.Row="0" Grid.Column="1" Grid.RowSpan="2">Button 0-1:1</Button>
 	<Button Grid.Row="0" Grid.Column="2">Button 0:2</Button>
 	<Button Grid.Row="1" Grid.Column="0">Button 1:0</Button>
 	<Button Grid.Row="1" Grid.Column="2">Button 1:2</Button>
 	<Button Grid.Row="2" Grid.Column="0">Button 2:0</Button>
 	<Button Grid.Row="2" Grid.Column="1" Grid.ColumnSpan="2">Button 2:1-2</Button>
</Grid>
